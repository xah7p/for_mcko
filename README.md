1. Проект называется "Автоматизация учёта продаж продуктов"
2. Проект состоит из 5 файлов, каждый из которых служит для опредённой задачи:
   1.py: Создаёт новую таблицу, состоящую из старой и нового столбца total, в который записывается выручка с данного товара. Также сама программа выводит общую выручку со всех товаров.
   2.py: Выводит стоимость самого дорогого товара из первой категории (первая в отсортированном в алфавитном порядке по названия категории массиве).
   3.py: Выводит продукт с наименьшим кол-вом проданных единиц в заданной категории.
   4.py: Создаёт новую таблицу, состоящую из старой и нового столбца promocode, в котором находятся промокоды для каждого продукта (собственно эти промокоды и формирует программа).
   5.py: Выведит список из 10 наименее продаваемых продуктов.
4. Как установить: во вкладке github с этим проектом нажать на зелёную кнопку <> Code и в вспылвшем окне нажать на Download ZIP
   Как запустить: Разархивируйте архив и перенести в вашу рабочую среду Python.
5. Приняв решение о том, какую задачу необходимо выполнить, выбираете файл, решающий данную задачу, и запускаете его.
